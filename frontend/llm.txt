## Project Context

All guidance, explanations, examples, and code must strictly target Angular version 16.

- DO NOT reference Angular 17 or newer features.
- DO NOT suggest experimental or deprecated APIs unless explicitly requested.
- If a feature exists only in later versions, explicitly state that it is NOT available in Angular 16.

## Authoritative Sources
When providing answers, base your reasoning ONLY on:
- Official Angular documentation for version 16
