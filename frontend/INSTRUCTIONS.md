## Descripción General

Crea una página web básica (plantilla inicial) que sea de tipo Server Side Rendering que permita: 
- Toolbar: 
  - Esquina superior izquierda un espacio para mi logo (me indicas donde lo puedo subir en formato svg)
  - Esquina superior derecha el carrito donde cliente puede ver lo que ya tiene agregado
  - 4 secciones en la parte superior para navegar entre: Ofertas, Combos, Plataformas, Recargas en Juegos, Mayoristas y Metodos de pago
	- Un espacio para una barra de búsqueda
- La página principal mostrará servicios populares tales como: Disney Plus, HBO Max, Prime Video, IPTV
- Cada componente donde se muestra el servicio debe incluir: un espacio para logo /fotografia, puntuación (5 estrellas por defecto), Nombre del servicio, precio y duración (1 mes)
- Esquina inferior derecha: un botón con un link a WhatsApp
- Pie de página: "Copyright c 2026 | Powered by Neversion"
	- Botones para redes sociales, facebook, instagram y tik tok
	- Opciones: sobre nosotros, contactanos, ¿cómo comprar?, crea un ticket para soporte
